# Apologetics

Static GitHub Pages apps that examine recurring weaknesses in Christian apologetics arguments.

## Apps

- Inductive Symmetry Audit: an interactive diagnostic for spotting cherry-picked inductive standards.

## GitHub Pages

This project is dependency-free and can be served from the repository root.

Recommended Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

For a local preview:

```sh
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
