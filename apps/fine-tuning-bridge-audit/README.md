# Fine-Tuning Bridge Audit

An upstream Crosshairs audit for checking what a fine-tuning argument is actually being asked to prove before it flows into thicker design, theism, or Christian claims.

The app focuses on:

- prior-commitment pressure
- explicit bridge premises from design to life-purpose, human-purpose, and theism
- beach/world-shape comparisons for sparse life versus abundant life
- target ambiguity, including nonhuman or unknown designer goals
- a guided handoff into `Theism Gradient`

## Run

```sh
python3 -m http.server 8080
```

Then open:

- `http://localhost:8080/apps/fine-tuning-bridge-audit/`

## Structure

- `index.html`: complete static audit page
- `app.js`: state, diagnostics, report generation, and Theism Gradient handoff
